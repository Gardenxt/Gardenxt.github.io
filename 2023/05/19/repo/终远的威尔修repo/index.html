终远的威尔修repo

发表于 2023-05-19 更新于 2025-07-27 分类于 游戏

感想

从3.24到今天通关，也有一部分绝望结局（席安）没有打。故事很好，下次不要再这么写了。

打完之后，最喜欢的角色应该是席安和安库。

一开始我还真的以为这个游戏是灵异方面的，毕竟出了一个死亡看守人，没想到居然全是科学，果然迷信是真的不行。结局也是，只有阿道夫的结局才算是好结局，跟其他人的结局不同，大家都还活着，虽然安库已经离世了。也不知道这样算不算真正意义上的离世。

根据官方推荐顺序攻略了，对于璐卡的结局印象深刻，完全没有好结局，每个结局都得死人，甚至救赎结局是双死，安库还会扫墓，两个人是得到救赎了，我反正绝望了。

安库也是，所有结局里都没有好结局，就算他是阿道夫我也希望他得到一个好结局，不然这也太难过了。受了那么多痛苦就为了见她开心的活着，在绝望结局中安库已经承受不住，阿道夫又穿越时空成为安库，反正怎么样安库都永远得不到好的结局了。
